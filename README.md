# Генератор постов для Threads (Grok / x.ai)

Локальное веб-приложение для ведения экспертного блогерского аккаунта в Threads.
В меню выбирается одна из трёх тем — **Менеджмент**, **Психология**, **Философия** —
и с помощью Grok (API console.x.ai) генерируется готовый к публикации пост.

## Возможности

- Меню выбора темы (3 тематики).
- Необязательное поле «пожелание к посту» (уточнение подтемы, стиля и т.д.).
- Генерация поста через Grok с ограничением под лимит Threads (500 символов).
- Счётчик символов, копирование в буфер, кнопка «Ещё вариант».

## Запуск

1. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

2. Создайте файл `.env` на основе `.env.example` и вставьте свой ключ x.ai:

   ```bash
   cp .env.example .env
   # затем впишите XAI_API_KEY (ключ с https://console.x.ai)
   ```

3. Запустите сервер:

   ```bash
   python app.py
   ```

4. Откройте в браузере: **http://127.0.0.1:5000**

## Настройки (`.env`)

| Переменная    | Назначение                                   | По умолчанию |
| ------------- | -------------------------------------------- | ------------ |
| `XAI_API_KEY` | Ключ API с console.x.ai (обязательно)        | —            |
| `XAI_MODEL`   | Модель Grok                                   | `grok-4.5`   |
| `HOST`        | Адрес локального сервера                      | `127.0.0.1`  |
| `PORT`        | Порт локального сервера                       | `5000`       |

## Структура

```
app.py             # Flask-сервер и API-роуты

```

> Файл `.env` с реальным ключом не коммитится (см. `.gitignore`).
